What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that allows developers to manipulate the structure, style, and content of a web page. It's a fundamental concept in web development and is essential for creating interactive and dynamic web pages.

How do you select an element from the DOM?

To select an element from the Document Object Model (DOM) in JavaScript, you use methods like getElementById to select by ID, getElementsByClassName to select by class name, getElementsByTagName to select by tag name, or querySelector and querySelectorAll to select using CSS selectors, which allows for more complex targeting based on attributes and combinations of selectors; each method returns the matching element(s) as a JavaScript object.

What is event delegation in the context of the DOM, and why is it useful?

In the context of the DOM (Document Object Model), "event delegation" is a technique where you attach a single event listener to a parent element, allowing you to handle events that occur on any of its child elements by leveraging the event bubbling mechanism;this is particularly useful when dealing with a large number of dynamically added elements, as it significantly improves performance by reducing the need to attach multiple event listeners to individual child elements

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM in JavaScript, you first need to select the desired element using methods like getElementById, querySelector, or getElementsByClassName, then access its properties like setAttribute to modify attributes and style to change CSS styles directly on the element.
